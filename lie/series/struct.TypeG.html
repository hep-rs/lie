The \(G_{n}\) exceptional Lie groups.

The only allows value of \(n\) is 2.

The Cartan matrix for \(G_{2}\) is:

\begin{equation} \begin{pmatrix} 2 & -1 \\ -3 & 2 \end{pmatrix} \end{equation}

Examples

use lie::RootSystem;
use lie::series::TypeG;

assert_eq!(g2.num_roots(), 14);

for r in g2.roots() {

}
